# Horizon TechX - DevOps Internship Tasks

This repository contains the completed tasks for the Horizon TechX DevOps Internship Program. The projects focus on modern DevOps practices, including containerization, CI/CD pipelines, and application deployment.

## Repository Structure

* **📁 [Task_4](./Task_4/) : Web Server using Docker**
  * Contains the `Dockerfile` and `index.html` used to build and deploy a custom, health-monitored Nginx web server. 
  * Explores container lifecycle management and deployment best practices.

* **📁 [Task_3](./Task_3/) : Java Application using Gradle** *(Coming Soon)*
  * Will contain the Java application source code and Gradle build configurations.
  * Focuses on automated builds and CI/CD integration.
